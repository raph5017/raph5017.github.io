Site Selection Tool for Identifying Areas of Biomass Production

Introduction & Methods

The aim of this study is to use python programming to create a site selection tool for identifying areas for biomass production. To test the tool, analysis was conducted on a biomass power station in south-west Scotland. The tool considers various factors to improve site selection and is created to be easily replicated at other locations. The github repository for this project can be found here. The README file explains the setup and installation.
